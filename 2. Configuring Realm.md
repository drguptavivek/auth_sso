# Copy the Keycloak themes jar from the container to your host

```bash
# docker cp auth_sso-keycloak-1:/opt/keycloak/lib/lib/main/org.keycloak.keycloak-themes-26.4.4.jar ./keycloak-themes.jar
# unzip keycloak-themes.jar -d keycloak-themes
# # Example: use keycloak.v2/login as base for your  login theme
# mkdir -p themes/myRealm
# cp -r keycloak-themes/theme/keycloak.v2/ themes/myRealm/
#  rm -r keycloak-themes
#  rm -r keycloak-themes.jar 
```

## NOW TWEK THE THEME

- themes/myRealm/login/theme.properties
- themes/myRealm/login/login.ftl
- themes/myRealm/login/resources/css/login.css

Add your logo to themes/myRealm/login/resources/img/aiims-logo.png


## USE TYhe Theme

 - Ensure docker-compose.yml mounts ./themes:/opt/keycloak/themes
 - Restart Keycloak
 - In realm  → Realm Settings → Themes, set Login Theme to myRealm
